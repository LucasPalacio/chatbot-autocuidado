# Como Contribuir

Obrigado pelo seu interesse em contribuir para o **Assistente de Autocuidado**!

## Passos para Contribuição
1. Faça um *Fork* deste repositório.
2. Crie uma branch com a sua feature: `git checkout -b minha-feature`
3. Faça as suas alterações e garanta que os testes e o linting passam (`pytest` e `flake8`).
4. Faça o *Commit* das suas mudanças: `git commit -m 'feat: minha nova feature'`
5. Faça o *Push* para a branch: `git push origin minha-feature`
6. Abra um *Pull Request* (PR).

Sua contribuição será revisada o mais rápido possível!
